## What is Encryptoni?

Encryptoni encrypts and decrypts files using **AES-256-GCM** authenticated encryption with **PBKDF2** key derivation (100,000 iterations, SHA-256). It supports three execution modes — serial, multiprocessing (`fork`), and multithreading (`pthread`) — achieving up to a **4× speedup** over serial on multi-core hardware.

---

## Features

| | |
|---|---|
| 🔒 **AES-256-GCM** | Authenticated encryption — detects any tampering during decryption |
| 🔑 **PBKDF2** | 100,000 iterations of HMAC-SHA-256 key stretching |
| ⚡ **4× Speedup** | Multiprocessing mode on a 1 GB benchmark (M2 Pro) |
| 💾 **Streaming I/O** | 64 KB chunks — handles files of any size without high memory use |
| 🛡️ **Zero Data Loss** | Atomic temp-file-and-rename — original files safe on interruption |
| 🖥️ **Cross-Platform** | macOS (arm64 / x86_64) and Linux (x86_64) |

## Quick Usage

### Encrypt a directory

```bash
encrypt ./my_folder --mode fork --password yourpass

decrypt ./my_folder --mode fork --password yourpass
```

### Single file

```bash
encrypt "./report.pdf" --password yourpass
decrypt "./report.pdf" --password yourpass
```

### Password options

```bash
# Flag
--password yourpass

# Environment variable
export ENCRYPTONI_PASSWORD=yourpass

# .env file in current directory
PASSWORD=yourpass
```

### Execution modes

| Mode | Flag | Description |
|---|---|---|
| Serial | `--mode serial` | Single-threaded (default) |
| Multiprocessing | `--mode fork` | Worker processes via `fork()` + shared memory — fastest |
| Multithreading | `--mode thread` | POSIX threads via `pthread` |

## Benchmarks

Tested on **Mac mini M2 Pro** — 1 GB dataset (200 × 5 MB files):

| Mode | Time | Speedup |
|---|---|---|
| Serial | 4s | 1.0× |
| Thread | 3s | 1.3× |
| **Fork** | **1s** | **4.0×** |

Multiprocessing wins because worker processes run in isolated address spaces, avoiding L2 cache bouncing and mutex contention.

## How It Works

Each encrypted file gets a **44-byte binary header** prepended:

```
[ Salt: 16 bytes ][ IV: 12 bytes ][ GCM Tag: 16 bytes ][ Ciphertext ]
```

- **Salt** — unique per file, drives PBKDF2 key derivation
- **IV** — unique per file, prevents ciphertext reuse
- **GCM Tag** — decryption fails immediately if the file is tampered with
